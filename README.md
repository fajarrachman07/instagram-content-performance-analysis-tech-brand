# Analisis Performa Konten Instagram Brand Teknologi

## Repository Outline
Repository ini berisi project analisis data yang membahas performa konten Instagram pada brand teknologi. Seluruh proses analisis dilakukan secara end-to-end, mulai dari pemahaman masalah bisnis, pengolahan data menggunakan Python, hingga penyajian hasil dalam bentuk dashboard visualisasi.

Isi repository ini terdiri dari:
1. notebook.ipynb – Notebook yang berisi proses analisis data menggunakan Python, mencakup data loading, data cleaning, eksplorasi data, visualisasi, serta analisis statistik.
2. clean_dataset.csv – Dataset hasil data cleaning yang digunakan untuk analisis dan pembuatan dashboard.
3. README.md – Dokumentasi project yang menjelaskan latar belakang, metode, dan output project.

## Problem Background
Instagram merupakan salah satu platform utama yang digunakan oleh brand teknologi untuk menjangkau audiens dan membangun brand awareness. Namun, performa setiap konten yang dipublikasikan dapat sangat bervariasi, baik dari sisi engagement maupun jangkauan. Tanpa analisis berbasis data, sulit bagi brand untuk memahami faktor apa saja yang memengaruhi performa konten serta karakteristik konten yang berpotensi viral. Oleh karena itu, project ini dibuat untuk menganalisis performa konten Instagram brand teknologi secara sistematis dan berbasis data.

## Project Output
Output dari project ini berupa analisis performa konten Instagram yang disajikan melalui Python Notebook serta dashboard interaktif menggunakan Tableau Public. Hasil analisis mencakup visualisasi performa konten, analisis statistik deskriptif dan inferensial, serta kesimpulan yang dapat digunakan sebagai dasar pengambilan keputusan strategi konten Instagram pada brand di bidang Teknologi.

## Data
Data yang digunakan dalam project ini merupakan dataset publik dari Kaggle yang berisi data performa konten Instagram, mencakup informasi jenis akun, jenis media konten, waktu posting, metrik interaksi (likes, comments, shares, dan saves), reach, impressions, serta indikator performa konten. Dataset telah melalui proses data cleaning dan difilter untuk fokus pada konten dengan kategori **Technology**, sehingga analisis yang dilakukan lebih relevan dengan konteks brand teknologi.

## Method
Metode yang digunakan dalam project ini meliputi exploratory data analysis (EDA) untuk memahami pola dan distribusi data, visualisasi data untuk membandingkan performa konten berdasarkan berbagai variabel, serta analisis statistik deskriptif dan inferensial. Uji ANOVA digunakan untuk menguji apakah terdapat perbedaan engagement rate yang signifikan secara statistik antar jenis media konten.

## Stacks
Project ini dikerjakan menggunakan Python sebagai bahasa pemrograman utama dengan library Pandas, Matplotlib, Seaborn, dan SciPy untuk analisis data. Proses analisis dilakukan menggunakan Jupyter Notebook, sedangkan visualisasi akhir disajikan dalam bentuk dashboard interaktif menggunakan Tableau Public.
